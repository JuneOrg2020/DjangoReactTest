# 関連情報紐づけアプリ

## アプリ概要

一つの情報（ワード）から連想される他の情報（ワード）を紐づけていきます。<br>
複数ユーザーで管理が可能です。<br>
情報と情報を紐づける「Linker」も定義できます<br>
